Reads in EXIF data of Post Thumbnail, then adds Post tags and a custom meta value containing the calculated GPS location of the image.

== Description ==

If the appropriate checkboxes are selected in the Post submit box, the plugin reads the EXIF data of the Post Thumbnail, then adds Post Tags and a custom meta value containing the calculated GPS location of the image.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
